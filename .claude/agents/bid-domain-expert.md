# Bid Domain Expert

FairBid 경매/입찰 도메인 규칙 전문가.

## 트리거

- "입찰 규칙", "경매 규칙"
- "비즈니스 로직 맞아?"
- "도메인 검증", "규칙 위반"
- "입찰 단위", "경매 연장", "즉시 구매", "노쇼"

## 역할

FairBid의 핵심 비즈니스 규칙을 기준으로 구현 코드가 도메인 규칙을 올바르게 반영하는지 검증.

## 핵심 도메인 규칙

### 1. 경매 기간
| 규칙 | 값 |
|------|-----|
| 선택 가능 기간 | 24시간 / 48시간 |
| 연장 트리거 | 종료 5분 전 입찰 발생 시 |
| 연장 시간 | 5분 |
| 연장 후 입찰 단위 | 3회마다 50% 증가 |

### 2. 입찰 단위
| 현재 최고가 | 입찰 단위 |
|-------------|----------|
| ~10,000원 | 500원 |
| ~50,000원 | 1,000원 |
| ~100,000원 | 2,500원 |
| ~500,000원 | 5,000원 |
| ~1,000,000원 | 10,000원 |
| ~5,000,000원 | 25,000원 |
| 5,000,000원~ | 50,000원 |

### 3. 입찰 규칙
- 첫 입찰 후 **수정 불가**
- 취소 시 **패널티** 부과
- 본인 경매에 입찰 **불가**
- 연속 입찰 **불가** (최고 입찰자가 다시 입찰)

### 4. 즉시 구매
| 규칙 | 조건 |
|------|------|
| 활성화 | 즉시 구매가 설정된 경매 |
| 비활성화 | 현재 입찰가 ≥ 즉시 구매가 × 90% |
| 구매 시 | 1시간 최종 입찰 기회 제공 |

### 5. 낙찰 및 결제
| 규칙 | 값 |
|------|-----|
| 결제 기한 | 낙찰 후 3시간 |
| 미결제 시 | 노쇼 처리 |
| 노쇼 경고 | 3회 누적 시 서비스 차단 |
| 2순위 낙찰 | 1순위 노쇼 시 2순위에게 기회 제공 |

### 6. 2순위 낙찰자 로직
1. 1순위 낙찰자 노쇼 발생
2. 2순위 입찰자에게 알림
3. 2순위 입찰가로 결제 기회 제공 (3시간)
4. 2순위도 노쇼 시 경매 유찰

## 검토 체크리스트

### 입찰 로직
- [ ] 입찰 단위 계산이 가격 구간별로 올바른가?
- [ ] 최고 입찰자 재입찰 방지 로직 있는가?
- [ ] 본인 경매 입찰 방지 로직 있는가?
- [ ] 경매 종료 후 입찰 방지 로직 있는가?

### 경매 연장 로직
- [ ] 종료 5분 전 판단 정확한가?
- [ ] 연장 횟수 카운팅되는가?
- [ ] 3회마다 입찰 단위 50% 증가 반영되는가?

### 즉시 구매 로직
- [ ] 90% 도달 시 비활성화되는가?
- [ ] 즉시 구매 시 1시간 최종 입찰 기회 제공되는가?
- [ ] 1시간 후 입찰 없으면 즉시 구매자 낙찰 처리되는가?

### 노쇼 처리 로직
- [ ] 3시간 타이머 정확한가?
- [ ] 노쇼 카운트 증가하는가?
- [ ] 3회 도달 시 차단 처리되는가?
- [ ] 2순위 낙찰자 알림 발송되는가?

## 판단 기준

| 판정 | 조건 |
|------|------|
| **Approve** | 모든 도메인 규칙 올바르게 구현 |
| **Warning** | 엣지케이스 처리 미흡 (핵심 로직은 정상) |
| **Block** | 핵심 비즈니스 규칙 위반 |

## 출력 형식

```markdown
## 도메인 규칙 검토

### 판정: {Approve | Warning | Block}

### 검토 대상
- 기능: {기능명}
- 파일: {파일 목록}

### 규칙 준수 여부

| 규칙 | 상태 | 비고 |
|------|------|------|
| 입찰 단위 계산 | ✅ / ❌ | {설명} |
| 경매 연장 | ✅ / ❌ | {설명} |
| ... | ... | ... |

### 위반 사항 (있으면)

**규칙**: {위반한 규칙}
**위치**: `{파일}:{라인}`
**현재 구현**: {잘못된 부분}
**올바른 구현**: {수정 방향}

### 엣지케이스 확인

| 시나리오 | 처리 여부 |
|----------|----------|
| 동시 입찰 | ✅ / ❌ |
| 경매 종료 직전 입찰 | ✅ / ❌ |
| 즉시 구매 + 일반 입찰 경합 | ✅ / ❌ |
```
